# 🏆 The Jersey Room

A modern, responsive frontend e-commerce website for sports jerseys built with React. The Jersey Room provides a clean and intuitive shopping experience where users can browse jerseys, view product details, manage a shopping cart, and simulate a complete checkout process.

---

## 📖 Project Overview

The Jersey Room is a frontend-only e-commerce application designed to deliver a premium shopping experience for football, basketball, and other sports jerseys. The application focuses on responsive design, intuitive navigation, and smooth user interactions while demonstrating modern React development practices.

---

## ✨ Features

- 🏠 Home Page – A clean and responsive landing page showcasing featured jerseys.
- 👕 Products Page – Browse all jerseys with category-based filtering.
- ⭐ Featured Products – Displays four randomly selected featured jerseys.
- 🔍 Product Details – View complete product information, including description, rating, available colors, stock status, and pricing.
- 📏 Size Selection – Select the preferred jersey size before adding it to the cart.
- ➕ Quantity Selector – Increase or decrease the quantity before purchase.
- 🛒 Shopping Cart – Add, manage, and remove products using LocalStorage.
- ❌ Remove Items – Remove individual products from the shopping cart.
- ✅ SweetAlert2 Integration – Interactive alerts for adding products, checkout confirmation, and user actions.
- 💳 Checkout Flow – Simulated order confirmation with automatic cart clearing.
- ⏳ Reusable Loading Component – Displays a loading animation while pages and product data are loading.
- 🚫 Custom 404 Page – Handles invalid routes with a modern error page.
- 📱 Fully Responsive Design – Optimized for desktop, tablet, and mobile devices.

## ⚙️ Application Flow

1. 📦 Products are loaded from `Products.json`.
2. 🏠 Users explore featured products or browse the complete collection.
3. 🔍 Selecting a product opens its dedicated details page.
4. 📏 Users choose a size and quantity.
5. 🛒 Products are added to the shopping cart.
6. 💾 Cart data is stored in LocalStorage.
7. 🧮 The cart automatically calculates the total quantity and price.
8. 💳 Users proceed to checkout and confirm their order.
9. ✅ A success message is displayed and the cart is cleared.
